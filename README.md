## Routes
Post: /api/v1/toppings <br/>
Get (all): /api/v1/toppings <br/>
Get (by id): /api/v1/toppings/:id <br/>
Put: /api/v1/toppings/:id <br/>
Delete: /api/v1/toppings/:id <br/>
